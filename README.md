# NI-2025-0160
Dataset from spatial learning experiments in Crenuchus spilurus. Individuals were tested in a visual-cue maze to assess navigation, learning, and cognitive flexibility across trials. Variables include sex, size, trial, sector, time to solve, and accuracy, allowing evaluation of sex-specific navigational strategies.

File Naming

The dataset is provided as a single Excel file containing all experimental observations.

File Content

The dataset contains the following variables:

id
Unique identifier assigned to each of the 50 tested individuals.

date
Date on which the experimental trial was conducted.

sex
Biological sex of the tested individual.

size
Standard length of the tested individual, measured in centimeters (cm).

trial
Experimental trial number conducted for each individual (1â€“5).

sector
Task sector evaluated during the trial.

time_to_solve
Time (in seconds) required to complete the task in each sector.

accuracy
Indicates whether the selected route matches the correct route (correct / incorrect).
